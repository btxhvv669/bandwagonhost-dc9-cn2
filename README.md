## 换 IP 之后要注意什么

换完 IP 之后，原来配置的所有服务都需要更新：防火墙规则、DNS 解析记录、SSH 客户端里保存的连接配置。尤其是如果你的域名解析到了旧 IP，记得去域名管理后台改过来，DNS 生效通常需要几分钟到几十分钟不等。

还有一个容易忽略的点：如果你之前在本地 known_hosts 文件里保存了旧 IP 的 SSH 指纹，连新 IP 的时候可能会报"警告"提示。把旧条目从 known_hosts 里删掉就好了，不是什么大问题。
