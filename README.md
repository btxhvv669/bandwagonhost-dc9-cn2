## DC9 CN2 是什么：一个机房 + 一条线路的组合

先说清楚基本概念。

**DC9**，全称是搬瓦工洛杉矶第九数据中心，机房代号 USCA_9，物理位置在美国洛杉矶。**CN2 GIA** 是中国电信旗下的高端国际出口线路，全称 ChinaNet Next Carrying Network，特点是优先级高、走独立通道、晚高峰不跟 163 普通骨干网抢带宽。

两者放在一起，**搬瓦工 DC9 CN2 GIA** 意思就是：这个洛杉矶机房跑的是电信 CN2 GIA 的回程线路，三网（电信、联通、移动）回程都走 GIA。

机房本身这两年也做了硬件升级，宿主机换成了 AMD EPYC CPU，硬盘改用 NVMe SSD，后台也标注出来了：`DC9 AMD+NVMe, CT CN2GIA, CMIN2, CUP`。所以如果你还在用旧版的方案，可以在 KiwiVM 面板里免费一键升级硬件。

## DC9 和 DC6 有啥区别，选哪个

这是搜 **搬瓦工 DC9 CN2** 的人最常见的疑问。

讲真，DC9 和 DC6 的线路质量在同价位下差距不大，但有一个技术层面的区别：

- **DC6（USCA_6）**：直接与洛杉矶本地运营商互联，出站带宽冗余相对更充裕，旁路流量处理更灵活
- **DC9（USCA_9）**：出站流量会先经过中国电信网络再转发，对电信用户非常友好，延迟表现也很稳

两者都接入了 CN2 GIA（电信/联通）+ CMIN2（移动）+ CUP（联通精品），三网晚高峰表现都不错。实际用下来，DC9 对纯电信用户来说有时候延迟略低；DC6 对三网兼顾的场景更稳定。
